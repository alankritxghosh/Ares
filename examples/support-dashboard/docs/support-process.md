# Support Process

## Current Workflow

- Tickets arrive from email, chat, and Zendesk.
- Support agents assign priority, update ticket status, and escalate unresolved issues to managers.
- Managers review open tickets twice a day and rebalance work when one agent has too many urgent cases.
- Customer satisfaction is reviewed weekly, but managers do not currently see response time and satisfaction in one place.

## Dashboard Needs

- Show counts by priority, status, region, and assigned agent.
- Highlight tickets with long response time or low satisfaction score.
- Make overloaded agents visible without requiring a spreadsheet export.
- Keep the first version focused on support managers rather than executives.
